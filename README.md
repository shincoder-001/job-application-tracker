# Job Application Tracker

A full-stack web application to track job applications.

## Tech Stack

Frontend:
- Next.js
- TailwindCSS

Backend:
- Node.js
- Express

Database:
- PostgreSQL (Neon)

ORM:
- Prisma

Authentication:
- JWT

Deployment:
- Frontend: Vercel
- Backend: Render
- Database: Neon

## Features

- Register / Login
- Add job applications
- Update status
- Dashboard
- Protected routes
